# Backtest Reports Directory

This directory stores visualization charts and reports.

Files are automatically created when using the Visualizer.

Chart types:
- `equity_curve_*.png` - Equity over time
- `drawdown_*.png` - Drawdown chart
- `trade_distribution_*.png` - P&L histogram
- `symbol_performance_*.png` - Performance by symbol

**Note:** Requires matplotlib to be installed
